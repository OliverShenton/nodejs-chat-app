 A dynamic chat application powered by Socket.io, enabling seamless real-time
                    messaging. Features include user location sharing, message timestamps, user
                    presence tracking, room-based messaging, automatic scrolling for new messages,
                    and user lists. The application is designed for a smooth and interactive
                    communication experience with a clean, responsive interface. Deployed for
                    accessibility and scalability.

This was created as part of the online course "https://www.udemy.com/course/the-complete-nodejs-developer-course-2/"
